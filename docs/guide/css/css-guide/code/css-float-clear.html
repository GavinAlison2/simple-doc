解决：通过添加clearfix类，容器能够正确包裹所有浮动元素，内容布局恢复正常。

清除浮动的CSS代码：

.clearfix::after {
    content: "";
    display: table;
    clear: both;
}

这是一种常用的清除浮动方法，通过在容器的::after伪元素上使用clear: both来清除浮动影响，使容器能够正确包裹所有浮动元素。
